FoodieBox

FoodieBox is a mobile food ordering application built with Flutter, designed to reduce food waste by enabling users to purchase leftover restaurant meals and near-expiry grocery items at discounted prices. The app provides a seamless experience for both users and vendors, supported by an efficient backend system.
